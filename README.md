# Avirup Mukherjee Portfolio

A high-performance, responsive portfolio website built with React and Vite. This project serves as a digital hub for showcasing research in Life Sciences and expertise in full-stack development. It features a unique Bio-CS aesthetic, combining stippled biological art with modern, interactive UI elements.

## Project Vision

The interface is designed to mirror the intersection of biology and computer science. It utilizes fluid motion, data-driven animations, and a tech-noir theme to present a professional narrative for academic and industrial opportunities, specifically targeting master's programs and research internships.

## Core Features

* **Dynamic Theme Engine:** A custom-built dark and light mode toggle that swaps between shadow-based stipple art (Light Mode) and highlight-based constellation stipple art (Dark Mode).
* **Antigravity Particle System:** A lightweight Canvas-based background animation that responds to mouse proximity, mimicking the movement of biological particles in a fluid medium.
* **Sticky Scroll Architecture:** Advanced scroll-linked transformations powered by Framer Motion, allowing the profile portrait to scale and move dynamically across different sections.
* **Interactive Contact Card:** A creative ATM Card style contact section with linked metadata for one-touch communication via email or telephone.
* **Responsive Fluid Design:** Fully optimized for all screen sizes, featuring a unique watermark layout for mobile devices.

## Tech Stack

* **Frontend:** React 18 (Vite)
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Graphics:** HTML5 Canvas API
* **Icons:** React Icons (Fa6)
